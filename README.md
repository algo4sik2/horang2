## Horang2 Coding

* 파이썬 자료구조 & 알고리즘 스터디  
* 기간: 2022.03.18 ~
* 시간: 월요일, 목요일 21:00 ~ 23:00 (주 2회, 2시간)  
* 교재: 파이썬 알고리즘 인터뷰(파알인), 파이썬으로 배우는 자료구조 핵심 원리
* 진행 방식:  

  * 진도에 맞게 각자 예습 및 과제
  * 예습한 내용으로 정규 스터디 시간에 코드 공유
  * 스터디 당일 문제 풀이 발표자 랜덤으로 선정
  * 스터디 시작 전 각자 새롭게 알게된 내용 공유 
  * 공유 자료는 노션, 디스코드, 깃허브 등에 업로드
